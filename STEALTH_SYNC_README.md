## 🎯 Быстрый старт (3 минуты)

### На домашнем ПК (один раз):

```cmd
# 1. Установите зависимости
cd backend && pip install -r requirements.txt
cd ../frontend && npm install && npm run build

# 2. Отредактируйте .env
# SYNC_PASSWORD=dandan

# 3. Запустите сервер
python backend/run.py

# Готово! https://192.168.1.100:443
```

### На рабочем ПК (каждый день):

```cmd
# Конец дня
cd C:\Users\YourName\Projects\MyProject
git add . && git commit -m "End of day"
backup_work_stealth.bat dandan

# Загрузите dump_ProjectName_*.dmp через браузер
# https://192.168.1.100:443/files

# Дома: нажмите "Sync Changes to Workspace"
```

## 🔑 Ключевые особенности

✅ **Инкрементальная синхронизация**
- Первый раз: полный проект (~50-200 MB)
- Потом: только изменения (~2-10 MB)

✅ **Зашифровано AES-256**
- Hidden headers (-mhe=on)
- Требует пароль для распаковки

✅ **Выглядит как memory dump**
- Расширение `.dmp`
- Имя файла: `dump_ProjectName_YYYYMMDD_HHMMSS.dmp`
- Маскирует истинное содержимое

✅ **OpSec задержки**
- Frontend: 3-5 сек перед отправкой
- Backend: 2-4 сек перед распаковкой
- Разрывает корреляцию "upload → process"

✅ **Автоматическое разрешение конфликтов**
- Проверка незакоммиченных изменений
- `git fetch` + `git reset --hard`
- Гарантирует точную копию

✅ **Привязка к проекту**
- Проверка PROJECT_NAME в имени файла
- Ошибка если мисматч
- Предотвращает синхронизацию неправильного проекта

## 📊 Размеры и производительность

| Сценарий | Размер | Время |
|----------|--------|-------|
| Первый полный бэкап | ~90% проекта | 5-30 мин |
| Ежедневная дельта | 1-50 MB | 1-5 мин |
| Еженедельная дельта | 10-100 MB | 5-15 мин |

**Пример:**
- Проект 500 MB → первый дамп ~50 MB (сжато)
- Ежедневные изменения → 2-5 MB

## 🔒 Безопасность

### Шифрование:
- AES-256 (военный стандарт)
- Hidden headers (-mhe=on)
- Требует пароль для распаковки

### Целостность:
- Git bundle (криптографически подписан)
- Проверка PROJECT_NAME
- Проверка конфликтов перед применением

### Конфиденциальность:
- HTTPS (TLS 1.3)
- Self-signed сертификат
- Локальная сеть (не интернет)

### Отсутствие следов:
- Временные файлы удаляются сразу
- Логи минимальны (SILENT_GIT)
- Выглядит как обычная работа

## 🆘 Если что-то не работает

### Проверьте:

1. **7-Zip установлен?**
   ```cmd
   "C:\Program Files\7-Zip\7z.exe" --version
   ```

2. **Пароль правильный?**
   ```env
   # .env
   SYNC_PASSWORD=dandan
   ```

3. **Вы в Git репозитории?**
   ```cmd
   git status
   ```

4. **Нет конфликтов?**
   ```cmd
   git status
   # Должно быть "nothing to commit"
   ```

5. **Проверьте логи:**
   ```
   Dashboard → System Log
   ```
